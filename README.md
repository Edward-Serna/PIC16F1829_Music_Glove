# Music Glove

> **Spring 2023 Microcomputer Systems I — Final Project | 🥉 3rd Place**

A wearable embedded music controller built around the **PIC16F1829** microcontroller. Flex sensors mounted on a glove map finger bends to musical tones, Morse code patterns, and signaling sequences — all output through a piezo buzzer with real-time serial feedback.

##  Features

- **4 flex sensor inputs** read via 10-bit ADC, one per finger
- **4 selectable modes** controlled over a UART serial terminal
- **Piezo buzzer tone generation** via software PWM (`__delay_us` toggling on RA4)
- **LED status indicators** on RA2, RA5, and RC6 for visual feedback
- **Serial debug output** at 9600 baud, monitor finger states and chord names in real time

---

## Modes

Select a mode at startup by typing its number (`1`–`4`) in a serial terminal after pressing `s` to begin.

| # | Mode | Description |
|---|------|-------------|
| 1 | **Beats** | Each finger triggers a distinct chord tone (C, D, E, G/F) |
| 2 | **DiffKey** | Alternate key mapping: fingers play A, B, C chords |
| 3 | **Morse** | Two fingers produce dots and dashes; third finger sends a newline |
| 4 | **Signal** | Yes/No/Emergency signals using colored LED + tone feedback |

## Hardware

| Component | Details |
|-----------|---------|
| Microcontroller | PIC16F1829 |
| IDE / Toolchain | MPLAB X IDE + XC8 Compiler |
| Oscillator | Internal 4 MHz (`OSCCON = 0x6A`) |
| Flex Sensors | 4× analog flex sensors (ADC channels 6, 7, 11, 13) |
| Output | Piezo buzzer on RA4 |
| LEDs | RA2 (red), RA5 (blue/green), RC6 (red) |
| Communication | UART — 9600 baud, RC4 (TX) / RC5 (RX) via `APFCON0` remapping |

### Pin Map

```
RC2 (AN6)  → Flex Sensor 1   RA4        → Buzzer (output)
RC3 (AN7)  → Flex Sensor 2   RA2        → LED 1 (output)
RB5 (AN11) → Flex Sensor 3   RA5        → LED 2 (output)
RC1 (AN13) → Flex Sensor 4   RC6        → LED 3 (output)
RC4        → UART TX          RC5        → UART RX
```

## Tone Reference

Notes are generated by toggling RA4 at the corresponding half-period delay:

| Note | Half-Period (`__delay_us`) | Frequency |
|------|---------------------------|-----------|
| A    | 2272 µs                   | ~220 Hz   |
| B    | 2024 µs                   | ~247 Hz   |
| C    | ~262 µs                   | ~523 Hz   |
| D    | ~294 µs                   | ~587 Hz   |
| E    | ~330 µs                   | ~659 Hz   |
| F    | ~349 µs                   | ~698 Hz   |
| G    | 392 µs                    | ~784 Hz   |
| Siren | 500 µs → 100 µs sweep   | Emergency alert |

## Getting Started

### Requirements
- [MPLAB X IDE](https://www.microchip.com/en-us/tools-resources/develop/mplab-x-ide)
- XC8 Compiler (free tier works)
- PICkit 3/4 or compatible programmer
- PIC16F1829 development board or breadboard circuit

### Build & Flash
1. Clone this repository
2. Open the project in MPLAB X (`File → Open Project`)
3. Connect your PICkit programmer
4. Click **Make and Program Device**

### Serial Monitor Setup
Connect to the PIC's UART at **9600 baud, 8N1** using any serial terminal (PuTTY, CoolTerm, MPLAB Data Visualizer).

```
Baud Rate : 9600
Data Bits : 8
Parity    : None
Stop Bits : 1
```

### First Run
1. Open the serial terminal
2. Reset the PIC, you'll see `Booting...`
3. Type `s` and press Enter to start
4. Select a mode (`1`–`4`) when prompted
5. Bend your fingers, ready to play some music!
